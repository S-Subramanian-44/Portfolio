Smart Gift Assistant


Developed a Smart Gift Assistant using Flask and MongoDB to provide personalized gift recommendations based on user interests, occasions, and budget. Integrated web scraping from Amazon to fetch live gift options and enhance user experience. Implemented optimization algorithms like Dynamic Programming, Greedy, and Backtracking to maximize value and handle real-time constraints.

Integrated modules for competitor tracking, seasonal trends, and financial projections to maximize box office success. Provided strategic recommendations based on Nash Equilibrium and real-time market conditions.
